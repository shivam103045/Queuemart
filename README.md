# Mini-project-
Smart Queue Management 
Definition:
A smart system that uses AI to detect people, monitor crowds, and manage queues in real time.
⚙️ How it works
Camera captures live video
YOLO detects and counts people
System checks crowd density & queue length
Data is stored in SQLite
Alerts shown if overcrowded or queue is long
🧠 Technologies Used
Python
YOLO (for person detection)
OpenCV (video processing)
SQLite (data storage)
🎯 Features
People counting
Queue length detection
Data storage & analysis
Real-time alerts
🧾 Conclusion
👉 This system uses YOLO for detection and SQLite for storing data, making it a simple and efficient mini project for crowd and queue management.
